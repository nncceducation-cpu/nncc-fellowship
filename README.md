# The Harvey Sarnat NNCC Fellowship — Website

A self-contained static website (HTML + CSS) for the University of Calgary Harvey Sarnat
Neonatal Neurocritical Care Fellowship.

## Preview it now
Double-click **index.html** to open the site in your browser. All pages and links work offline
(photos and web fonts need an internet connection).

## Pages
- `index.html` — Home
- `fellowship.html` — Program & Curriculum
- `competencies.html` — Competencies (CanMEDS framework)
- `assessment.html` — Assessment & Portfolio
- `governance.html` — Governance, Policies & People
- `innovation.html` — Innovation in Education Lab
- `styles.css` — shared styling for every page

## Put it online
See **DEPLOYMENT-GUIDE.md** for step-by-step instructions covering:
- Namecheap domain + Netlify (easiest)
- Namecheap domain + GitHub Pages (free, with version history)

## Make it fully self-contained (optional)
The photos load from Thinkific's CDN by default. To bundle them into this folder, run the
script in `images/` (see `images/README.txt`).

## Editing
Open any `.html` file in a text editor to change wording. The look-and-feel lives entirely in
`styles.css`. No build step, server, or database is required.
